# seelove_data_web（看见爱情的模样）
See what love looks like.

看见爱情的模样。

## 项目简介 🥰
通过大数据采集，对网易云关于爱情的评论采集分词之后通过云图展示。让我们通过大数据见识到爱情的模样

## 项目架构 🧰
运行环境：Linux, Python3, nodejs

框架：React

特别鸣谢：@ant-design/charts

## 项目结构 🧀
```
.
├── data (数据采集目录)
├── public (前端静态资源)
├── scripts (Python 脚本，大数据采集，大数据清洗)
└── src (前端可视化代码)
```
